# Mincomi Test Site

This is a small static site to test uploading and hosting.

How to test locally:
- Open index.html in your browser (double-click) or use Live Server.
- Click the "Click me" button and submit the contact form to see demo responses.

How to host online (quick):
- Push this folder to a GitHub repository and enable GitHub Pages (branch: main, folder: /).
- Or drag-and-drop this folder to Netlify Drop or upload to Vercel.

If you want, I can push to GitHub and enable Pages for you.
